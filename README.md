# Robo-Advisor
Machine-Learning based robo-advisor
<br>
- Spring framework 
- Java+Jsp+Mybatis 
- Oracle 11g 
- R serve

## Main Functions
- Financial planning
- House-Price Prediction by Arima and Forcast in R
- Financial-Products Recommendation by UBCF(User-based Collaborative Filtering) in R
